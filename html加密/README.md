# 📄 `html加密.html` 

# 🔐 核心 HTML 加密工具 · 文件密钥解锁

一个纯前端 HTML 加密器，可将任意 HTML 文件加密生成「解锁页面」和「密钥文件」。只有持有正确 `secret.key` 的用户才能解密查看原始内容。

## ✨ 核心功能

- **AES-256-CBC 加密**：使用随机生成的密钥和 IV
- **SHA-256 密钥校验**：解锁时验证密钥文件哈希，防止错误文件解密
- **一键生成双文件**：
  - `unlock.html` – 解锁入口页面
  - `secret.key` – 密钥文件（请妥善保管）

## 🚀 使用方法

### 🔒 加密方

1. 打开 `给html加密.html`
2. 选择你想要保护的 HTML 文件（如 `核心.html`）
3. 点击「开始加密并生成文件」
4. 自动下载 `unlock.html` 和 `secret.key`

### 🔓 解密方

1. 打开 `unlock.html`
2. 上传 `secret.key` 文件
3. 验证通过后，原始 HTML 内容自动渲染显示

## 🧰 技术栈

- CryptoJS – AES 加密 / SHA-256 哈希
- Web Crypto API (用于计算文件哈希)

## 🔒 安全性说明

- 密钥文件为 64 位十六进制字符串，采用 AES-256 加密
- 解锁页面仅包含密文和 IV，无密钥无法解密
- 所有运算均在本地浏览器完成，无数据上传

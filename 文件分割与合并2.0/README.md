# 📦 文件分卷/合并工具（极简版）

一个轻量级纯前端工具，可将大文件分割成指定大小的分卷，并能通过总配置文件一键合并还原。特别适合绕过某些平台的文件大小限制。

## ✨ 核心功能

- **文件分卷**：自定义分卷大小（MB），自动生成编号分卷 + 总配置文件
- **一键下载**：生成后可将所有分卷打包下载
- **极简合并**：只需选择总配置文件和所有分卷（放在同一文件夹），自动识别并合并
- **纯前端**：所有处理均在浏览器完成，文件不上传服务器

## 🚀 使用方法

### 分割文件

1. 选择要分割的文件
2. 设置分卷大小（默认 10 MB）
3. 点击「生成分卷文件」
4. 点击「一键下载所有文件」保存分卷及配置文件

生成文件示例：
```
001_视频.mp4
002_视频.mp4
视频.mp4.vol.json
```

### 合并文件

1. 将**所有分卷**和**总配置文件**放在同一个文件夹
2. 点击「选择总配置文件 + 同目录分卷」，选中该文件夹
3. 点击「开始自动合并」，浏览器自动下载还原后的原文件

## 🧰 技术栈

- 原生 JavaScript (Blob / File API)

## ⚠️ 注意事项

- 合并时必须保证分卷完整且命名未被修改
- 分卷命名格式为 `序号_原文件名`，配置文件为 `原文件名.vol.json`

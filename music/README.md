# 🎵 磨砂音乐 · 智能搜索 + 横向卡片

一款设计精美的纯前端音乐播放器，通过加载 GitHub Gist 中的 JSON 歌单来播放音乐。支持歌词滚动、搜索过滤、自动加载链接。

## ✨ 核心功能

- **Gist 歌单加载**：粘贴 Raw URL 即可获取歌单（JSON 格式）
- **横向卡片布局**：歌曲列表采用现代化毛玻璃卡片设计
- **实时搜索**：输入关键词快速过滤歌名/艺术家
- **歌词同步**：支持 LRC 格式歌词，当前行自动高亮并居中
- **播放模式**：列表循环 / 单曲循环 / 随机播放
- **自动加载**：粘贴链接后自动拉取歌单（防抖）

## 🚀 使用方法

1. 点击右上角齿轮 ⚙️，粘贴 Gist Raw URL（或点击「使用演示 Gist」体验）
2. 歌单加载后点击任意歌曲开始播放
3. 右侧面板展示封面、歌词，支持进度/音量调节

## 📦 歌单格式要求

Gist 文件中应包含一个 JSON 数组，每个歌曲对象格式如下：

```json
[
  {
    "title": "歌曲名",
    "artist": "艺术家",
    "url": "音频直链",
    "pic": "封面图 URL",
    "lrc": "歌词文件 URL (可选)"
  }
]

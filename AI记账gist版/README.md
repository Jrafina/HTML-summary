# 💳 多卡 AI 记账 · GitHub Gist 自动同步

一个极简但强大的个人记账工具，纯前端实现，数据通过 GitHub Gist 自动云端同步。支持多张银行卡余额管理、AI 语义解析记账、月度统计图表。

## ✨ 核心功能

- **AI 语义解析**：输入「招商银行信用卡买早餐 25 元」自动识别金额、银行卡、收支类别
- **多银行卡管理**：每张卡独立初始金额，消费/收入自动更新余额
- **全自动 Gist 同步**：
  - 输入 GitHub Token 后自动拉取/上传数据
  - 记账、修改、删除均实时同步云端
- **月度统计**：总收入/支出、日均支出、每日柱状图（Chart.js）
- **移动端适配**：底部导航栏，手指友好交互

## 🚀 使用方法

1. 打开页面，点击底部「同步」页，输入 GitHub Personal Access Token（需 `gist` 权限）
2. 返回「银行卡」页，添加你的信用卡/储蓄卡并设置初始余额
3. 在「记账」页输入自然语言描述，点击「AI 解析并记账」
4. 数据自动保存至 Gist，换设备登录相同 GitHub 账号即可同步

## 🔧 数据存储

- 本地：`localStorage`
- 云端：GitHub Gist（文件名为 `account_book.json`）

## 🧰 技术栈

- 原生 JavaScript (ES6)
- Chart.js – 统计图表
- GitHub Gist API v3

## 📌 示例记账语句

```
招商银行信用卡午餐 48 元
工资收入 15000 元
交通银行储蓄卡交房租 3200 元
```

## ⚠️ 注意事项

- GitHub Token 仅保存在你的浏览器中，不会上传到任何第三方服务器
- 首次使用会自动创建一个私有 Gist 存储数据
